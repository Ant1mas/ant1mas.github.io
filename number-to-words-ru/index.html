number-to-words-ru node module - convert number to russian words

number-to-words-ru

Модуль Node.js для конвертации числа в слова на русском языке.
Ваше число

Делителем числа может быть ' . ' или ' , ' или ' / '

Результат

Минус восемнадцать миллионов четыреста пятьдесят одна тысяча триста два рубля 01 копейка

Опции

Рубль

Округление отключено

Код


    const numberToWordsRu = require('number-to-words-ru');

    numberToWordsRu.convert('-18451302.01');

Установка

Установить с помощью npm:

npm install number-to-words-ru

Установить с помощью yarn:

yarn add number-to-words-ru
